جاوا اسکرپٹ کا تعارف

آئیے دیکھتے ہیں کہ JavaScript کے بارے میں کیا خاص بات ہے، ہم اس کو استعمال کر کے کیا حاصل کر سکتے ہیں، اور کونسی دوسری ٹیکنالوجیز اس کے ساتھ اچھی کارکردگی کا مظاہرہ کرتی ہیں۔

JavaScript کیا ہے؟

جاوا اسکرپٹ کو ابتدائی طور پر "ویب صفحات کو زندہ کرنے" کے لیے بنایا گیا تھا۔ javascript میں پروگراموں کو سکرپٹ کہتے ہیں۔ انہیں web pageکے ایچ ٹی ایم ایل(HTML) میں لکھا جا سکتا ہے اور صفحہ لوڈ ہوتے ہی خود بخود چلایا جا سکتا ہے۔ انہیں چلانے کے لیے خاص تیاری یا compilation کی ضرورت نہیں ہے۔

info اسے جاوا اسکرپٹ کیوں کہا جاتا ہے؟

جب جاوا اسکرپٹ بنایا گیا تو اسے اصل میں Mocha کہا جاتا تھا، جس کا نام بدل کر LiveScript رکھا گیا، اور پھر اس کا نام JavaScript رکھ دیا گیا۔
